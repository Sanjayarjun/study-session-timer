# 📚 Smart Study Session Timer with Focus Stats

A productivity web app built with **Python & Streamlit** that helps students manage study sessions, track focus time, and visualize progress.

---

## 🚀 Features
- ⏳ Start a customizable study session timer
- 📊 Real-time progress tracking
- 📈 Focus statistics visualization with charts
- 🌍 Deployed live with Streamlit Cloud

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- Matplotlib  

---

## 🌍 Live Demo
👉 [Try the app here](https://sanjayarjun-study-timer.streamlit.app/)

---

## 📂 Project Structure
- `streamlit_app.py` → Main Streamlit app  
- `requirements.txt` → Dependencies  

---

## 📜 How to Run Locally
```bash
# Clone repo
git clone https://github.com/Sanjayarjun/study-session-timer.git

# Navigate into folder
cd study-session-timer

# Install dependencies
pip install -r requirements.txt

# Run Streamlit app
streamlit run streamlit_app.py
